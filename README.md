# Robotics

This repository contains the materials for the Robotics discipline, part of the Artificial Intelligence Specialization course.

## Week 1: Introduction to Robotics

The first week of the course provides an introduction to the fundamental concepts of Robotics, including its history, components, and the different types of robots.

### Topics Covered:

*   **Introduction to Robotics:** Fundamental concepts of robotics, its historical evolution, definitions, and the essential components that constitute a robotic system.
*   **Types of Robots: Manipulators:** A look at manipulator robots (robotic arms), their structure, joints, and the difference between serial and parallel manipulators.
*   **Types of Robots: Mobile:** An overview of mobile robots, including terrestrial (wheeled and legged), aerial, and aquatic vehicles.
*   **Classification of Robots by Level of Autonomy:** Understanding the different levels of robot autonomy, from fully controlled to fully autonomous.

### Files:

*   `week_1/1_Introduction.md`: Overview of the fundamental concepts of robotics.
*   `week_1/2_Types_of_Manipulator_Robots.md`: Detailed notes on manipulator robots.
*   `week_1/3_Types_of_Mobile_Robots.md`: An explanation of the different types of mobile robots.
*   `week_1/4_Types_of_Robots_Autonomy.md`: A description of the levels of autonomy in robots.

## Week 2: Sensors and Actuators

The second week of the course delves into the essential components that allow a robot to perceive its environment and act upon it: sensors and actuators.

### Topics Covered:

*   **Introduction to Sensors and Actuators:** An overview of the importance and classification of sensors and actuators in robots.
*   **Proprioceptive Sensors:** Detailed notes on proprioceptive sensors, which measure the internal state of the robot.
*   **Exteroceptive Sensors:** An explanation of exteroceptive sensors, which collect information from the environment.
*   **Actuators:** A description of actuators, the components responsible for robot movement.

### Files:

*   `week_2/1_Introduction_to_Sensors_and_Actuators.md`: Overview of sensors and actuators.
*   `week_2/2_Proprioceptive_Sensors.md`: Detailed notes on proprioceptive sensors.
*   `week_2/3_Exteroceptive_Sensors_Part_1.md`: An explanation of exteroceptive sensors (Part 1).
*   `week_2/4_Exteroceptive_Sensors_Part_2.md`: A continuation of the explanation of exteroceptive sensors (Part 2).
*   `week_2/5_Actuators.md`: A description of actuators.

## Week 3: Robotic Architectures, Perception, and Localization

The third week focuses on the fundamental architectures that govern robotic systems, along with key concepts in perception and localization.

### Topics Covered:

*   **Architectures for Robotic Systems:** Understanding different architectural paradigms for designing and implementing robotic systems.
*   **Perception - Introduction:** An introduction to how robots perceive their environment, covering various sensing modalities and their importance.
*   **Localization - Dead Reckoning:** Detailed explanation of dead reckoning as a method for robot localization, including its principles and limitations.
*   **Localization - Kalman Filter:** Introduction to the Kalman Filter and its application in robot localization for state estimation and uncertainty reduction.
*   **Localization - Particle Filter:** Exploration of the Particle Filter (Monte Carlo Localization) for robust robot localization in complex environments.

### Files:

*   `week_3/1_Architectures_for_Robotic_Systems.md`: Overview of robotic system architectures.
*   `week_3/2_Perception_Introduction.md`: Introduction to robotic perception.
*   `week_3/3_Localization_Dead_Reckoning.md`: Explanation of dead reckoning for localization.
*   `week_3/4_Localization_Kalman_Filter.md`: Introduction to Kalman Filter for localization.
*   `week_3/5_Localization_Particle_Filter.md`: Description of Particle Filter for localization.

## Week 4: Mapping

The fourth week explores various techniques robots use to create and maintain maps of their environment.

### Topics Covered:

*   **Mapping - Types of Maps:** Overview of different types of maps used in robotics, such as occupancy grids, feature-based maps, and topological maps.
*   **Mapping - Exploration:** Concepts and strategies for autonomous exploration, enabling robots to discover unknown environments and build maps.
*   **Mapping - SLAM:** Introduction to Simultaneous Localization and Mapping (SLAM), a fundamental problem in robotics where a robot builds a map and localizes itself within it concurrently.

### Files:

*   `week_4/1_mapping_map_types.md`: Overview of different map types.
*   `week_4/2_mapping_exploration.md`: Concepts of robotic exploration.
*   `week_4/3_mapping_slam.md`: Introduction to SLAM.

## Week 5: Kinematic Control

The fifth week delves into the principles and methods of kinematic control for robots, focusing on how robots achieve desired movements.

### Topics Covered:

*   **Kinematic Control - Introduction:** Introduction to kinematic control, distinguishing it from dynamic control and its importance in robot motion planning.
*   **Kinematic Control - Concepts of Control Theory:** Fundamental concepts from control theory relevant to kinematic control, such as feedback, stability, and error correction.
*   **Kinematic Control - Control for Wheeled Robot:** Application of kinematic control principles specifically to wheeled mobile robots, including differential drive and omnidirectional robots.
*   **Kinematic Control - Code for Wheeled Robot Control:** Practical implementation aspects and code examples for controlling wheeled robots based on kinematic models.

### Files:

*   `week_5/1_Introduction_to_Kinematic_Control.md`: Introduction to kinematic control.
*   `week_5/2_Concepts_of_Control_Theory.md`: Concepts of control theory for kinematic control.
*   `week_5/3_Control_for_Wheeled_Robot.md`: Control strategies for wheeled robots.
*   `week_5/4_Code_for_Wheeled_Robot_Control.md`: Code examples for wheeled robot control.

## Week 6: Navigation

The sixth week covers the essential aspects of robot navigation, including path planning and obstacle avoidance.

### Topics Covered:

*   **Navigation - Introduction:** Introduction to robot navigation, defining its components: localization, mapping, path planning, and motion control.
*   **Navigation - Path Planning with Deterministic Solutions:** Exploration of deterministic path planning algorithms, such as Dijkstra's and A* algorithms, for finding optimal paths in known environments.
*   **Navigation - Code for Path Planning with Deterministic Solutions:** Practical implementation and code examples for deterministic path planning algorithms.
*   **Navigation - Path Planning with RRT:** Introduction to the Rapidly-explored Random Tree (RRT) algorithm for path planning in high-dimensional or complex spaces.

### Files:

*   `week_6/1_Navigation_Introduction.md`: Introduction to robot navigation.
*   `week_6/2_Path_Planning_with_Deterministic_Solutions.md`: Deterministic path planning solutions.
*   `week_6/3_Path_Planning_Code_with_Deterministic_Solutions.md`: Code for deterministic path planning.
*   `week_6/4_Path_Planning_with_RRT.md`: Path planning using RRT algorithm.
